# 零序 - 小手机网页应用规格文档

## 1. 项目概述
- **项目名称**: 零序 (Xiaoshouji)
- **类型**: 模拟手机界面的角色扮演聊天WebApp
- **核心功能**: 调用AI大模型API进行角色扮演聊天
- **目标用户**: 中文用户群体

## 2. UI/UX 规格

### 2.1 整体布局
- **模拟手机容器**: 375px × 812px (iPhone X比例)，居中显示
- **状态栏**: 顶部44px，显示时间和电池
- **底部安全区**: 34px
- **圆角**: 40px
- **阴影**: 0 20px 60px rgba(0,0,0,0.15)

### 2.2 色彩系统
```
--primary: #98D8C8 (薄荷绿主色)
--primary-dark: #7BC4B0 (深薄荷)
--primary-light: #B8E6D9 (浅薄荷)
--bg-screen: #F5FBF8 (屏幕背景)
--bg-card: #FFFFFF (卡片背景)
--text-primary: #2D3436 (主文字)
--text-secondary: #636E72 (次要文字)
--accent: #F8B500 (点缀色-金黄)
--danger: #FF6B6B (危险/删除)
--border: #E8F0EC (边框色)
```

### 2.3 字体
- **主字体**: "Noto Sans SC", "PingFang SC", sans-serif
- **标题**: 18px, font-weight: 600
- **正文**: 14px, font-weight: 400
- **小字**: 12px

### 2.4 启动页 (Splash Screen)
- 全屏薄荷绿渐变背景
- 中央显示"零序"Logo文字
- 1.5秒后自动淡出进入桌面

### 2.5 主屏幕 (桌面)
- **页面数量**: 2页（左右滑动）
- **桌面图标网格**: 4列 × 4行
- **图标尺寸**: 60px × 60px
- **图标间距**: 20px
- **应用名称**: 图标下方，12px，白色/深色文字带阴影

### 2.6 底部常驻栏
- 高度: 50px + 安全区
- 4个等宽装饰占位图标
- 图标上显示"占位"文字
- 无实际功能

## 3. 功能规格

### 3.1 千风集 (聊天App)
**底部导航栏**:
- 千风 (聊天列表) - 图标: 消息气泡
- 同行 (通讯录) - 图标: 联系人
- 浮世 (朋友圈) - 图标: 地球
- 余韵 (个人中心) - 图标: 人物

**千风页 - 聊天列表**:
- 好友列表，每行显示: 头像、名称、最后消息、时间
- 点击进入聊天界面

**聊天界面**:
- 顶部返回栏 + 好友名称
- 消息气泡: 对方(左侧薄荷绿)、自己(右侧白色)
- 底部输入框 + 发送按钮
- 发送消息调用AI API获取回复

### 3.2 同行者 (角色管理)
- 角色列表页面
- 新增角色按钮
- 角色编辑表单:
  - 姓名 (输入框)
  - 头像 (选择预设头像或上传)
  - 人设 (多行文本区域)
- 删除角色功能

### 3.3 枢机室 (AI配置)
**基础配置**:
- Base URL 输入框
- API Key 输入框
- Model Name 输入框

**预设配置管理**:
- 预设列表显示
- 新增预设按钮
- 预设编辑: 名称 + 上述三项参数
- 选择当前预设
- 删除预设

**进阶功能**:
- 拉取模型列表按钮
- 温度参数: 0-2 滑块
- 记忆条数: 1-50 输入框

### 3.4 世界书
- 占位页面，显示"即将上线"文字

## 4. 动画规格
- 页面切换: 300ms ease滑动
- 图标点击: 100ms scale(0.95)缩放反馈
- 启动页淡出: 500ms fadeOut
- 底部导航切换: 200ms渐变

## 5. 数据存储
所有数据使用LocalStorage:
- `zero_config`: AI配置参数
- `zero_presets`: 预设配置列表
- `zero_roles`: 角色列表
- `zero_chats`: 聊天记录
- `zero_current_preset`: 当前选中的预设ID
